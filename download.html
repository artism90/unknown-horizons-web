Linux

We provide packages for many linux distributions:

Ubuntu & Linux Mint users should install Unknown Horizons via this PPA.

Add the PPA:

sudo add-apt-repository ppa:christoph-egger/unknown-horizons

Install Unknown Horizons:

sudo apt-get update && sudo apt-get install unknown-horizons

Debian users can find Unknown Horizons in the official repos:

sudo aptitude install unknown-horizons

Arch Linux users can find Unknown Horizons in the AUR.

Fedora and OpenSUSE users can find Unknown Horizons packages in the OpenSUSE software directory.
